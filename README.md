## 대회 방식

- 제한시간: 3시간
- 사용 도구: VS Code + GitHub Copilot 보이스 코딩
- 기술 스택: 웹 앱
- 사용 언어: Python 3.12, JavaScript (node.js LTS 22.x), Java 21, .NET 9 중 선택
- 데이터베이스: 메모리 DB 또는 SQLite
